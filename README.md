### 使用说明
``` javascript
$ npm install -g rexhang-cli
```

``` javascript
$ rex-cli -h
$ rex-cli init      || i
$ rex-cli add       || a
$ rex-cli delete    || d
$ rex-cli list      || l
$ rex-cli update    || u  
````
